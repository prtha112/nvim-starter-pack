### Config foler
```sh
cd ~
mkdir .config
mkdir nvim
touch init.vim
```

## Requirement
1. Python3
2. Pip3
3. Rust
4. Rustup
5. Cargo
6. Node
7. Yarn
8. arch -arm64 brew install ctags

## Install 
```sh
:PlugInstall
```

## Clean Plugin
```sh
:PlugClean
```
